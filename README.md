# Calculadora de Matrizes

Uma calculadora de matrizes matemáticas:  
**[Teste aqui](https://gabrielsantos15.github.io/Matriz/)**

---

## 🚀 **Funcionalidades**  
- Cálculo do determinante de qualquer matriz 3x3.  
- Análise de condição de alinhamento de 3 pontos.  

---

## 📚 **Base Teórica**  
O projeto foi desenvolvido para uma atividade escolar de matemática no IFSP (Instituto Federal de São Paulo)
